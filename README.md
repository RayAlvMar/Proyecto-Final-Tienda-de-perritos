# Proyecto-Final-Tienda-de-perritos
En este proyecto final llevaremos a cabo una tienda de adopción, artículos y venta de perritos.

Alvarez Martinez Raymundo
Monreal Rosas Adrian Alexis
4-D

Este proyecto resuelve un problema que lleva mucho tiempo existiendo el cual son los perritos callejeros y como estos son vulnerables y muchas veces peligrosos debido al entorno callejero en el que viven, gracias al sistema de adopciones estos perritos tienen una mejor vida y una familia puede ser feliz con ellos, y a su vez nuestros productos resuelven el problema de cuan costoso o complicado puede llegar a ser el conseguir articulos para los perros.

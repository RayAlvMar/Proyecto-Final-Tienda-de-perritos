DoggyWorld

Sistema web para gestión de perros en adopción, compra y cuidado responsable.

Propósito del proyecto

DoggyWorld centraliza información de perros disponibles para adopción o compra, ayudando a tomar decisiones responsables.

Funcionalidad

Registro de usuarios, login y catálogo de perros almacenados en MongoDB.

Base de datos

MongoDB con colecciones de usuarios y perros para gestión dinámica.

Tecnologías

  • Flask
  • MongoDB
  • HTML / CSS / Bootstrap
  • Jinja2

Seguridad

Sesiones y contraseñas encriptadas para proteger usuarios.

Enfoque

Promover adopción responsable y bienestar animal usando tecnología.
